PREGUNTAS FRECUENTES

  • ¿Qué diseño presenta el Razer DeathAdder V2 Pro?

    El Razer DeathAdder V2 Pro presenta una forma ergonómica única para diestros.

  • ¿El Razer DeathAdder V2 Pro tiene click de desplazamiento horizontal?

    No, el Razer DeathAdder V2 Pro no incluye un click de desplazamiento horizontal.

  • ¿Cómo puedo reasigno los botones y personalizo la configuración del Razer DeathAdder V2 Pro?

    Para poder reasignar los botones, las macros y los efectos de iluminación, entre otros, primero tendrás que descargar e instalar Razer Synapse 3.

  • ¿El Razer DeathAdder V2 Pro es compatible con la base de carga de mi Razer Viper Ultimate o mi Razer Basilisk Ultimate?

    Sí. Puedes usar la base de carga para recargar la batería de tu mouse y ampliar el alcance del dongle USB inalámbrico.

  • ¿Para qué sirve el botón en la parte inferior del Razer DeathAdder V2 Pro?

    Por defecto, este botón te permite cambiar entre los diferentes perfiles guardados en el mouse. Con Razer Synapse 3, puedes cambiar la configuración de este botón, como si fuera cualquier otro del mouse.
